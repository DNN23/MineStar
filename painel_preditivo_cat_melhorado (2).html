Formato esperado dos arquivos

Eventos — colunas: Frota, Máquina, Data da Ocorrência_, Descrição do alarme, Nível do evento, Operador, TAG (Sistema é opcional, derivado se vazio).

Frota → CAT_785C, CAT_777G Data → dd/mm/aaaa hh:mm:ss Nível → 1, 2 ou 3 (3 = crítico)

Dicionários CID / FMI / MID (opcionais) — três arquivos separados, detectados pelo cabeçalho. Pode selecionar os três de uma vez.

CID.csv → CID, Descrição FMI.csv → Descrição, FMI MID.csv → MID, Descrição
